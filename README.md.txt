## Description
A full-stack beauty and skincare shopping app built using React Native (Expo) for frontend and Node.js for backend.

## Features
- View beauty products
- Product detail page
- Add to cart (UI only)
- Simple and responsive layout

## Tech Stack
- Frontend: React Native (Expo)
- Backend: Node.js, Express
- Database: MongoDB (optional)

## How to Run
1. cd frontend
2. npm install
3. npx expo start
